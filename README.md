# HappyQQT-ASM
An Intel 8086 Emulator for QQTang-ASM.

Game resource except of the title picture comes from [Tencent QQTang](https://qqtang.qq.com/).

![](./pic/title.png)
![](./pic/screenshot1.png)

## Control 
- Move the player
    - WASD
- Enter game
    - space

## Requirement
- G++ with pthread
- OpenGL
    - OpenGL
    - GLUT
    - GLU

## Build
```bash
make
```

## Run
```bash
./main
```
