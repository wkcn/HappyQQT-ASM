# 欢乐QQ堂 ASM版 ver 1.0

除了封面为自己绘图，其余游戏素材来自腾讯游戏“QQ堂”

# 操作方式

方向键移动

空格开始游戏，或者在结束游戏时返回开始界面

# 剧情

年兽来糖果王国踢足球了，你要打败它！

界面上方为年兽的血量，共5 HP

你可以踢足球:-)

在驱逐年兽的同时，要避免足球炸弹的伤害！

# 编译方式
在Linux系统下输入sudo make

强烈推荐使用VirtualBox或VMware Workstation虚拟机以保持游戏的流畅性

make run: 使用qemu运行

make bochs: 使用bochs运行 （极不推荐，只能用来调试）
